Для установки проекта Laravel Framework 5.8.36 понадобится:
1) Склонировать проект из репозитория GIT.
2) Настройки сервера (виртуального):
    - MySQL-5.7
    - Apache-PHP-7
    - PHP-7.1
3) Создать БД с названием "gl".
4) Запустить миграции для создания таблиц в БД "gl": в консоли открыть путь к корневой папке проекта. Выполнить команду "php artisan migrate".
5) Открыть проект в браузере.

Пояснения по проекту:
1) / - главная стр с пунктами меню
2) /add - стр для добавления новых Паст
3) /create - стр добавляющая 1 Пасту в БД (сделано для быстрого наполнения БД. По пути App\Paste.php в методе create() можно задать значения заполняемых полей)
4) /list - страница выводящая все записи из БД (сделано для удобства)
5) /single/{какой-то-определенный-хэш} - стр отображающая 1 определенную Пасту
